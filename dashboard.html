WHAT IS TRENDPULSE?
TrendPulse is a real-time YouTube trend analytics dashboard. It pulls live data directly from YouTube Data API v3 to show you what the world is watching — broken down by country, category, and content type (Videos vs Shorts).
TOOLS & SECTIONS
🔥 Trending Videos
Top 10 long-form videos per category, ranked by views. Filter by region and category from the topbar.
📱 Trending Shorts
Top 20 YouTube Shorts (≤60 seconds) shown in a native vertical portrait grid. Opens directly on youtube.com/shorts.
📊 Trend Charts
Views-over-time line chart and category doughnut — filterable by category and duration type (Videos / Shorts / Both).
📺 Top Channels
Which channels appear most in trending. Ranked by total views. Filter by category and content type.
🔖 Bookmarks
Save any trending video or Short for later reference. Bookmarks persist across sessions.
